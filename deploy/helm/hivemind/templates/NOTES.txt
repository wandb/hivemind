1. Watch the rollout:

     kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "hivemind.fullname" . }}-api

   On first install the api stays NotReady until the schema migration Job
   finishes:

     kubectl -n {{ .Release.Namespace }} logs job/{{ include "hivemind.fullname" . }}-migrate-{{ .Release.Revision }}

2. Reach the dashboard:
{{- if .Values.gateway.enabled }}

     {{ .Values.externalURL }}   (HTTPRoute host: {{ if .Values.gateway.hostnames }}{{ first .Values.gateway.hostnames }}{{ else }}{{ include "hivemind.externalHost" . }}{{ end }})

   Ensure that host resolves to your Gateway's address. TLS is terminated on
   the Gateway listener (owned by your platform team), not by this chart.
{{- else if .Values.ingress.enabled }}

     {{ .Values.externalURL }}   (Ingress host: {{ include "hivemind.ingressHost" . }})

   Ensure that host resolves to your ingress controller's address.
{{- else }}

     kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "hivemind.fullname" . }}-api {{ .Values.api.service.port }}:{{ .Values.api.service.port }}
     open {{ .Values.externalURL }}
{{- end }}

3. Finish setup: visit {{ .Values.externalURL }}/setup to create a GitHub App
   (or run single-player). See docs/self-hosting.md.
